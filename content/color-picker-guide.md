# 颜色选择器完整指南：HEX/RGB/HSL/CMYK 在线转换与配色方案生成

## 什么是颜色选择器？

颜色选择器（Color Picker）是一种帮助用户可视化地选取颜色并获取其多种编码格式的工具。无论是网页设计师、UI/UX 设计师、前端开发者还是平面设计师，在日常工作中都需要频繁使用颜色选择器来获取精确的颜色值。

## 五种主流颜色格式详解

### HEX（十六进制颜色码）

HEX 颜色码以 `#RRGGBB` 格式表示，每个通道取值范围 00-FF（0-255）。它是 Web 开发中最常用的颜色表示方式，简洁且广泛支持。例如 `#e11d48` 代表一种鲜艳的红色。

**HEX 格式特点：**
- 6 位十六进制数 + # 前缀
- 支持缩写（如 `#e11` 等于 `#ee1111`）
- 支持 RGBA（如 `#e11d48ff`，最后两位代表透明度）
- 被所有浏览器原生支持

### RGB（红绿蓝色彩模型）

RGB 色彩模型是最底层、最物理的颜色表示方式。它通过红（R）、绿（G）、蓝（B）三种基色的不同强度混合出所有颜色，每个通道范围 0-255。

**实际应用：**
- CSS 中的 `rgb(225, 29, 72)` 直接控制显示颜色
- `rgba(225, 29, 72, 0.5)` 可额外控制透明度
- LED 屏幕、显示器、手机屏幕均基于 RGB 原理

### HSL（色相/饱和度/明度）

HSL 是最符合人类直觉的颜色模型，也是设计师最常使用的格式。

| 参数 | 范围 | 说明 |
|------|------|------|
| H（色相） | 0-360° | 决定颜色类型，红=0/360，绿=120，蓝=240 |
| S（饱和度） | 0-100% | 0%为灰色，100%为最鲜艳 |
| L（明度） | 0-100% | 0%为黑色，100%为白色 |

**为什么设计师偏爱 HSL？** 因为调整一个参数就能产生直觉可预测的变化：色相改变颜色类型，饱和度改变鲜艳度，明度控制整体亮度。

### HSV/HSB（色相/饱和度/明度值）

HSV 与 HSL 非常相似，但 V（明度值）的定义不同。在 HSV 中，V 值代表颜色中的最大亮度分量，而 HSL 的 L 是平均亮度。大多数图形软件（Photoshop、Figma等）的取色器使用 HSB/HSV 模型。

### CMYK（青/品红/黄/黑）

CMYK 是印刷行业的色彩标准，与 RGB 完全相反：RGB 是加色混合（叠加光线产生颜色），CMYK 是减色混合（吸收光线产生颜色）。

- C（Cyan）= 青色
- M（Magenta）= 品红
- Y（Yellow）= 黄色
- K（Key/Black）= 黑色

**注意：** 屏幕设计的 RGB 颜色在印刷时可能完全不同，这就需要在设计时用 CMYK 预览做检查。

## 配色方案：颜色搭配的艺术

### 互补色（Complementary）

色环上相距 180° 的颜色互为互补色。例如红色与青色、蓝色与橙色。互补色搭配产生强烈的视觉对比，适合需要突出重要元素的设计。

### 邻近色（Analogous）

色环上相邻的 2-4 个颜色（相距约 30°）构成邻近色搭配。例如蓝色、蓝绿色、绿色。这种搭配和谐统一，适合需要营造整体氛围的设计。

### 三色配色（Triadic）

色环上均匀分布（相距 120°）的三个颜色构成三色配色。例如红、绿、蓝三原色。这种搭配色彩丰富但不杂乱，适合需要活泼风格的设计。

### 四色配色（Tetradic / Rectangle）

色环上均匀分布的四种颜色（相距 90°）构成四色配色。色彩组合最为丰富，需要谨慎使用以避免过于杂乱。

## 颜色选择器的使用场景

### 1. 网页开发
前端开发中，从设计稿提取颜色、调整颜色值、确保无障碍对比度都需要颜色选择器。

### 2. UI/UX 设计
设计界面时，创建符合品牌规范的颜色系统、测试不同配色方案、生成次色和强调色。

### 3. 数据可视化
选择可区分的颜色序列来标记图表中的数据系列，确保各种设备上都能清晰辨认。

### 4. 平面设计
将数字设计转化为印刷品时，需要 CMYK 颜色值以确保色彩准确还原。

### 5. 品牌设计
建立品牌色彩规范，定义主色、次色、辅助色的准确色值，在不同媒介上保持一致。

## 在线颜色选择器使用教程

### 基本操作
1. **色相选取**：拖动色相滑块（0-360°）选择颜色基调
2. **精调颜色**：在饱和度/明度面板中点击或拖动，调整颜色的鲜艳度和明暗
3. **实时预览**：大色块实时显示当前选择颜色
4. **复制色值**：点击目标格式旁的复制按钮，一键复制到剪贴板

### 高级功能
1. **手动输入**：直接在 HEX、RGB、HSL 输入框中输入已知色值，取色器自动同步
2. **配色方案生成**：一键生成互补色、邻近色、三色、四色等配色方案
3. **多种格式互转**：HEX ↔ RGB ↔ HSL ↔ HSV ↔ CMYK 五格式全自动互转
