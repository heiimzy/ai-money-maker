# 正则表达式测试器 (Regex Tester)

免费在线正则表达式测试工具，支持 JavaScript/PCRE 语法。

## 功能特点

- 🔎 **实时匹配** — 输入即匹配，结果高亮显示
- 🎯 **捕获分组** — 自动识别并展示所有捕获组
- 🔄 **替换模式** — 支持 `$1`、`$2` 等反向引用替换
- 📊 **匹配统计** — 匹配数量、耗时、文本长度实时统计
- 📖 **正则速查表** — 内置常用正则语法参考
- 🌙 **暗色主题** — 支持亮色/暗色切换
- 📱 **响应式设计** — 支持手机、平板、桌面端
- 🔒 **隐私安全** — 所有数据仅在本地浏览器处理

## 使用说明

1. 输入正则表达式（不需要两边的 `/`，Flags 单独选择）
2. 输入测试文本（支持示例数据加载）
3. 匹配结果实时高亮
4. 点击 Ctrl+Enter 快速测试

## 技术实现

- 纯前端 HTML/CSS/JS，无任何后端依赖
- JavaScript `RegExp` 引擎实现正则匹配
- Canvas-free，原生 DOM 操作
- CSS 变量实现主题切换
