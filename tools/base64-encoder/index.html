💡 什么是 Base64 编码？

Base64 是一种用 64 个可打印字符（A-Z、a-z、0-9、+、/）来表示二进制数据的编码方式。它将二进制数据转换为 ASCII 字符串，常用于在文本协议（如 HTTP、JSON、HTML）中传输二进制数据。

🔄 编码 vs 解码

编码 (Encode)：将普通文本或二进制数据转换为 Base64 格式。解码 (Decode)：将 Base64 格式还原为原始文本或数据。

🎯 常见使用场景

  • 数据 URL (Data URL)：在 HTML/CSS 中直接嵌入图片或字体
  • JWT Token：JSON Web Token 使用 Base64 编码载荷
  • API 传输：在 JSON/XML 中传输二进制数据
  • 邮件附件：MIME 协议使用 Base64 对附件编码
  • 认证信息：HTTP Basic Auth 使用 Base64 编码用户名和密码

⚠️ 注意事项

Base64 不是加密算法，它只是数据格式的转换，没有密钥保护。不要用 Base64 来保护敏感信息。真正的安全需要 AES、RSA 等加密算法配合使用。
