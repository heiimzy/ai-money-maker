# Base64 在线编码解码工具

免费在线 Base64 编码/解码工具，支持文本编码、文本解码、文件编码三种模式。

## 功能特性

- **文本编码**：将普通文本编码为 Base64 格式
- **文本解码**：将 Base64 字符串解码为可读文本
- **文件编码**：选择文件（图片、PDF、JSON 等）编码为 Base64（Data URL）
- **暗色/亮色主题**：支持主题切换并持久化用户偏好
- **一键复制**：点击结果区域或手动复制按钮
- **快捷键支持**：Ctrl+Enter 快速编码/解码
- **拖拽上传**：支持文件拖拽上传
- **响应式设计**：完美适配移动端和桌面端

## 在线体验

部署后可通过以下地址访问：
- `https://<your-domain>/base64-encoder/`

## 技术栈

- 原生 HTML5 + CSS3 + JavaScript (ES6)
- 使用 `btoa()` / `atob()` 进行 Base64 转换
- 使用 `FileReader` API 实现文件读取
- CSS 变量实现主题切换
- 无外部依赖
