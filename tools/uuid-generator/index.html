什么是 UUID/GUID？

UUID（Universally Unique Identifier，通用唯一识别码）是一种标准的128位标识符，用于在分布式系统中唯一标识信息。GUID（Globally Unique Identifier）是微软对UUID标准的一个实现，两者在实际使用中基本可以互换。

UUID 的版本

本工具默认生成 UUID v4（随机生成），这是最常用的版本。其他常见版本包括：

  • UUID v1 — 基于时间戳和MAC地址生成
  • UUID v3 — 基于MD5哈希和命名空间
  • UUID v4（本工具）— 完全随机生成，安全性最高
  • UUID v5 — 基于SHA-1哈希和命名空间

UUID 的常见应用场景

数据库主键：分布式系统中使用自增ID容易产生冲突，UUID可以确保全局唯一性。许多现代数据库（PostgreSQL、MongoDB）对UUID有原生支持。

API 资源标识：RESTful API中的资源ID通常使用UUID，避免暴露自增ID带来的信息泄露风险。

会话标识：Web应用中的Session ID、CSRF Token等安全相关的标识符，使用UUID v4可以保证不可预测性。

文件命名：上传文件时使用UUID重命名可以避免文件名冲突和路径遍历攻击。
