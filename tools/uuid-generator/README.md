免费在线 UUID v4 随机生成工具。支持批量生成、多种格式（大小写/有无连字符）、一键复制。

## 功能特性

- ✅ **UUID v4 随机生成** — 使用浏览器 Crypto API 生成真正的随机 UUID
- ✅ **批量生成** — 一次最多生成 100 个 UUID
- ✅ **格式切换** — 支持大写/小写、带连字符/无连字符
- ✅ **一键复制** — 点击单个 UUID 或一键全部复制
- ✅ **暗色模式** — 支持日间/夜间主题切换
- ✅ **快捷键** — Ctrl+Enter 快速生成
- ✅ **响应式设计** — 完美适配手机和桌面
- ✅ **隐私保护** — 所有运算在浏览器本地完成，数据不上传服务器
