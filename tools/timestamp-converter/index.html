💡 什么是时间戳 (Timestamp)？

Unix 时间戳（Unix Timestamp）是指从 1970年1月1日00:00:00 UTC（协调世界时）开始经过的秒数（或毫秒数）。它是计算机系统中通用的时间表示方式，广泛应用于：数据库时间字段、API 响应时间、日志记录、JWT Token 过期时间、文件时间戳等场景。

🛠️ 如何使用这个工具？

时间戳 → 日期时间：在输入框中粘贴 Unix 时间戳（秒级如 1715712000 或毫秒级如 1715712000000），工具会自动检测格式并显示对应的 UTC 时间、本地时间和 ISO 8601 格式。

日期时间 → 时间戳：输入日期时间字符串（如 2024-05-15 10:00:00），工具会同时计算出秒级和毫秒级时间戳。

批量转换：每行输入一个时间戳，一键批量转换为日期时间。

⏰ 常见时间戳格式

秒级 (Seconds)：10位数字，如 1715712000 — 最常用的 Unix 标准格式

毫秒级 (Milliseconds)：13位数字，如 1715712000000 — JavaScript 和其他编程语言常用

ISO 8601：2024-05-15T10:00:00Z — 人类可读的国际化标准格式
