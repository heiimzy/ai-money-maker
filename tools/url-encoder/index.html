💡 什么是 URL 编码 (Percent Encoding)？

URL 编码（又称百分号编码）是一种将 URL 中的特殊字符转换为 %XX 格式的编码方式。URL 中只允许使用 ASCII 字母、数字和一些特殊字符（-_.~），其他字符（如空格、中文、#、&、? 等）必须经过编码才能在 URL 中安全传输。

例如：空格 → %20（或 +），中文"你好"→ %E4%BD%A0%E5%A5%BD

📋 常见编码对照表

空格 → %20 或 +

# → %23 — & → %26 — ? → %3F

= → %3D — / → %2F — : → %3A

@ → %40 — % → %25 (编码本身)
