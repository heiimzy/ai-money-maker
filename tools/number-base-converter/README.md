# Number Base Converter (进制转换器)

在线进制转换工具，支持二进制、八进制、十进制、十六进制及自定义进制（2-36位）之间的相互转换。

## 功能特点
- 🔢 支持 4 种标准进制 + 自定义进制（基数 2-36）
- 🔄 实时转换，输入即出结果
- 📋 一键复制各进制结果
- 🔀 任意进制之间互转
- 🌙 暗色/亮色主题切换
- ⌨️ Ctrl+Enter 快捷键快速转换
- 📱 响应式设计，移动端友好
- 🔒 所有数据仅在本地浏览器处理

## 使用方法
1. 在"输入"区域选择输入数字的进制并输入数字
2. 选择目标进制
3. 结果实时显示（或按 Ctrl+Enter 手动转换）
4. 点击"复制"按钮复制结果到剪贴板
