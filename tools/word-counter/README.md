# 在线字数统计工具 (Word Counter)

一个美观、免费的在线字数统计工具，支持中英文混排统计。

## 功能
- 实时统计：总字符数、中文字数、英文单词数、行数、段落数、空格数、标点数
- 复制统计结果
- 加载示例文本
- 响应式设计，支持移动端

## 技术栈
- 纯静态 HTML + CSS + JavaScript
- 所有计算在浏览器本地完成，无服务器端
- SEO 优化：meta tags, Open Graph, semantic HTML

## 部署
- 可部署到 GitHub Pages / Vercel / Netlify
- 预留了广告位（顶部、中间、底部）

## 盈利模式
- 流量变现：通过 Google AdSense 或 百度联盟投放广告
- SEO 关键词：字数统计、在线字数统计、字符统计、word counter
