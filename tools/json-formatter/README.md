# JSON 格式化/校验工具

一个功能完整、美观的在线 JSON 格式化与验证工具。

## 功能

- ✅ **JSON 格式化** - 美化压缩的 JSON，带 2 空格缩进
- ✅ **JSON 验证** - 检查 JSON 语法正确性，错误时定位到具体行和列
- ✅ **JSON 压缩** - 去除所有空白字符，减小体积
- ✅ **复制结果** - 一键复制格式化后的 JSON
- ✅ **加载示例** - 快速加载示例 JSON 数据
- ✅ **暗色/亮色主题** - 跟随系统或手动切换
- ✅ **键盘快捷键** - Ctrl+Enter 快速格式化
- ✅ **响应式设计** - 移动端和桌面端均适配
