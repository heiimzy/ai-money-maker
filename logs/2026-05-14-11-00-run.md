## 经验教训
1. **CSS 变量重构**是支持主题切换的最佳方案 — 一次性定义好 `:root` 和 `[data-theme="dark"]`，所有元素自动适配
2. **工具间交叉链接**是 SEO 的重要策略 — 建立站内链接网络，提升搜索引擎对 site 的收录和权重
3. **UUID 生成器**使用 Crypto API (`crypto.getRandomValues`) 比 `Math.random()` 更安全，是真随机数
4. **批量复制功能**体验很重要 — 点击单个 UUID 复制带视觉反馈，按行分隔全部复制
5. **数量限制**避免滥用 — 设置 max=100 的合理边界
